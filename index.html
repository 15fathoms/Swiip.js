SWIIP.JS

WHY
SWIIP.JS

LIGHTWEIGHT

Swiip.js is a lightweight carousel library written in vanilla JavaScript. It offers a straightforward and customizable solution for implementing carousels on web pages.

EASY INTEGRATION

Swiip.js is built with simplicity in mind. With just a few lines of code, you can have a fully functional carousel up and running on your website. Its lightweight nature ensures fast loading times, improving overall user experience.

FLEXIBLE

Swiip.js offers a range of customizable options, allowing you to tailor the carousel to match your specific needs. From defining the number of visible slides to adjusting transition speed and easing, you have full control over the carousel's behavior.

RESPONSIVE

Swiip.js seamlessly adapts to different screen sizes and devices. It provides built-in responsive capabilities, enabling your carousel to gracefully resize and reposition slides to ensure optimal display across various devices, from desktops to mobile devices.

NO DEPENDENCIES

Swiip.js is a standalone library with no external dependencies. You don't need to worry about additional libraries or frameworks, keeping your project clean and free from unnecessary dependencies.

BASIC
USAGE

1. Include Swiip.js

<script src="https://cdn.jsdelivr.net/gh/EmmanuelCartelli/Swiip.js@master/swiip.js"></script>
<link href="https://cdn.jsdelivr.net/gh/EmmanuelCartelli/Swiip.js@master/swiip.css">

2. Create HTML markup

<div class="container">
    <div class="item">
        <img src="img1.jpg">
    </div>
    <div class="item">
        <img src="img2.jpg">
    </div>
    <div class="item">
        <img src="img3.jpg">
    </div>
    <div class="item">
        <img src="img4.jpg">
    </div>
</div>

3. Initialize Swiip.js

const container = document.querySelector('.container');
const swiip = container.Swiip();

2. Navigation Buttons

const container = document.querySelector('.container');
const swiip = container.Swiip({
    prevBtn: document.querySelector('.prev'),
    nextBtn: document.querySelector('.next')
})

3. Custom options

const container = document.querySelector('.container');
const swiip = container.Swiip({
    slidesVisible : 3,
    slidesToScroll : 1,
    autoPlay : true,
    autoplaySpeed : 2000,
    gap: 2,
    responsive : [
        {
            breakpoint: 1024,
            slidesVisible: 2,
            slidesToScroll: 2,
            gap: 2,
        },
        {
            breakpoint: 768,
            slidesVisible: 1,
            slidesToScroll: 1,
            gap: 1,
            autoplay: false,
        }
    ]
})
